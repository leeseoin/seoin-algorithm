# 큐 (Queue)

## 한 줄 요약
**먼저 들어온 게 먼저 나가는** (FIFO, First In First Out) 자료구조. 줄 서서 기다리는 거랑 같음.

## 동작
```
enqueue(1)  →  [1]
enqueue(2)  →  [1, 2]
enqueue(3)  →  [1, 2, 3]
dequeue()   →  1 반환, [2, 3]
dequeue()   →  2 반환, [3]
front()     →  3 (제거 X, 보기만)
```

들어온 순서대로 나감. 스택과 정반대.

## 파이썬 구현 — `collections.deque` 필수

```python
from collections import deque

q = deque()

q.append(1)           # enqueue → O(1)
q.append(2)
q.append(3)
print(q.popleft())    # 1 (가장 앞에서 빼기) → O(1)
print(q[0])           # front
print(len(q))
print(not q)          # 비었는지
```

### 왜 `list` 안 쓰나?

```python
arr = [1, 2, 3]
arr.pop(0)            # 1 반환, but O(n)!
```

`list.pop(0)`은 **모든 원소를 한 칸씩 앞으로 옮겨야** 해서 O(n). 큐 동작에 list 쓰면 **시간 초과**나기 쉬움. 큐가 필요하면 **무조건 `deque`**.

## deque의 위력

```python
from collections import deque
dq = deque([1, 2, 3])

dq.append(4)          # 오른쪽 추가, O(1)
dq.appendleft(0)      # 왼쪽 추가, O(1)
dq.pop()              # 오른쪽 제거, O(1)
dq.popleft()          # 왼쪽 제거, O(1)
dq.rotate(1)          # 회전 (오른쪽으로 1칸)
dq.rotate(-1)         # 왼쪽으로 1칸
```

양방향 큐(deque) — **양끝 모두 O(1)**. 큐도 되고 스택도 됨.

## 시간복잡도

| 연산 | `deque` | `list` |
|---|---|---|
| 끝 추가 | O(1) | O(1) |
| 끝 제거 | O(1) | O(1) |
| 앞 추가 | O(1) | O(n) ❌ |
| **앞 제거** | **O(1)** | **O(n)** ❌ |
| 인덱스 접근 | O(n) | O(1) |

`deque`는 인덱스 접근이 느린 게 약점이지만, 큐로 쓸 땐 인덱스 접근 안 하니까 무관.

## 언제 쓰나 — 대표 패턴

### 1. BFS (너비 우선 탐색) ⭐⭐⭐
가장 중요한 활용. 그래프/격자 탐색의 기본.
```python
from collections import deque

def bfs(start, graph):
    visited = {start}
    q = deque([start])
    while q:
        node = q.popleft()
        for nxt in graph[node]:
            if nxt not in visited:
                visited.add(nxt)
                q.append(nxt)
```

### 2. 캐시 / LRU
"오래된 거부터 버리기" — 앞에서 빼고 뒤에 넣기.

### 3. 작업 대기열 / 시뮬레이션
프로세스 스케줄링, 줄 서기 시뮬레이션

### 4. 슬라이딩 윈도우
일정 크기의 윈도우를 옮기면서 양끝 처리. **`deque`의 양끝 O(1)** 이 핵심.

### 5. 회전 / 원형 배열
`deque.rotate()` 한 줄.

## 우선순위 큐는 별개

이름은 "큐"지만 동작이 다름. **가장 작은(또는 큰) 값이 먼저 나옴**. 다익스트라, top-K 문제에 사용.

→ `heapq` 모듈 (`알고리즘/2순위_다익스트라.md` 참고)

## 실전 팁

- 큐 = `deque`. 외워두기.
- BFS 짤 때 `q.append(start)` `q.popleft()` 패턴은 손에 익어야 함.
- 초기값 줄 때 `deque([1, 2, 3])` 식으로 iterable 전달.
- 비었는지 검사 `if not q:` 가 깔끔.
